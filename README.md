# Kian's Market Dashboard 📈

Daily futures snapshot for NQ, ES, and RTY — auto-updated Mon–Fri at 14:00 UTC.

## What it shows
- **Fear & Greed Index** (alternative.me)
- **NQ** — Nasdaq 100 Futures
- **ES** — S&P 500 Futures  
- **RTY** — Russell 2000 Futures
- Daily bias signals + 5-day mini charts
- Auto-generated daily brief

## Setup (GitHub Pages)

1. Fork or push this repo to GitHub
2. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
3. Done — live at `https://yourusername.github.io/kian-dashboard/`

## Auto-updates

GitHub Actions runs the data fetcher Mon–Fri at 14:00 UTC.  
You can also trigger it manually from the **Actions** tab.
